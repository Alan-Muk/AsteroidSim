 Asteroid Simulation (MPC Data)

 Features

* Uses real MPCORB data
* 3D interactive visualization with Plotly
* Animated asteroid motion over time
* Automatically downloads latest dataset

 Installation

```bash
git clone https://github.com/YOUR_USERNAME/asteroid-simulation.git
cd asteroid-simulation
pip install -r requirements.txt
```

 Run

```bash
python main.py
```

 Data Source

Minor Planet Center (MPC)

 Notes

* Default runs ~5000 asteroids for performance
* Increase limit in code if your system allows
